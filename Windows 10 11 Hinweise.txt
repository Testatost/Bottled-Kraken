Bottled Kraken v3.1 - Windows 10/11 Hinweise
==============================================

Empfohlene Entwicklungsumgebung:
- Python 3.11 bis 3.13, 64-bit
- Virtuelle Umgebung im Projektordner

Installation aus dem Quellcode:

    py -3.13 -m venv .venv
    .venv\Scripts\activate
    python -m pip install --upgrade pip setuptools wheel
    python -m pip install -r requirements.txt

Start:

    python main.py

Build mit PyInstaller:

    python -m PyInstaller --clean --noconfirm main.spec

Hinweis zu GPU-Backends:
Die Standard-App bleibt eine CPU-Onefile-Version. NVIDIA CUDA kann aus dem Programm heraus über Optionen -> CPU/GPU -> CUDA (NVIDIA) installieren als externes Backend installiert werden. Das Backend wird unter %%LOCALAPPDATA%%\BottledKraken\backends\ installiert.
